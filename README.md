# KC, Kemono and Coomer crossplatform desktop notifications

## Current features:
- Persistent memory
- Creator search
- Creator selection
- Desktop notifications
- GUI

## Upcoming features:
- Settings menu
    - Check-timing customization
    - Colour customization
- File download
- Taskbar icon
    - Minimize to taskbar
- Link notifications
- Scroll-to-load
- Advanced search
    - Post search
    - Tag search
- Authentication
